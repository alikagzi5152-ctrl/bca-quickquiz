## Abstract
QuickQuiz is a lightweight web application that presents a 5-question multiple-choice quiz to users. The app scores answers and displays results. It demonstrates use of DOM manipulation, event handling and basic CSS styling.

## Objectives
- Build a functional quiz using vanilla JavaScript.
- Demonstrate understanding of HTML structure and CSS styling.
- Prepare submission-ready source code and documentation.

## Tools & Technologies
- HTML5, CSS3, JavaScript (ES6)
- Any modern web browser

## Features
- Start/Retry quiz flow
- Progress tracker
- Immediate feedback on answers (correct/wrong)
- Final score display

## Installation
1. Download project files.
2. Open `index.html` in a browser.

## Code structure
- `index.html` — layout and references
- `styles.css` — design
- `script.js` — questions array and logic

## Testing
Tested on Chrome, Firefox and Edge (desktop). Mobile layout responsive for basic use.

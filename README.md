# QuickQuiz — BCA Minor Project (5th Sem)

## Project summary
QuickQuiz is a small multiple-choice web quiz built with plain HTML, CSS and JavaScript.  
It demonstrates front-end fundamentals and is suitable for a minor project submission.

## Files in repository
- `index.html` — main web page
- `styles.css` — styling
- `script.js` — quiz logic (questions, scoring)
- `REPORT.md` — project report (summary, installation, screenshots)
- `SOURCE_CODE.pdf` — (optional) PDF containing full source code (generated if environment allowed)
- `submission_instructions.md` — how to prepare GitHub repo and create source code PDF

## How to run locally
1. Extract the zip.
2. Open `index.html` in any modern browser.
3. No server is required; this is a static site.
